Build static weblog with coleslaw

标签：Coleslaw

发布于 2023-03-14

有人说过，在这个世界上，使用静态博客写博客的人比写博客的人还要多。如果你是软件人，也会懂得这个笑话。在本文中，我将讨论如何使用 Coleslaw 构建静态博客。

静态博客

静态博客是与世界分享个人想法、观点和经验的一种流行的方式。它易于创建和维护，并可根据作者的风格和喜好进行自定义。这不，多少都有人用过 Jekyll，但在 Windows 上安装起 Ruby 来并不容易，尤其是在不科学的情况内，也有不少人使用 Hugo，因为它快，以及 Gatsby、Hexo、GitBook，甚至用过更小众的 mdBook、Pelican、Zola，但极少人会使用 Frog、Hakyll，以及 Coleslaw。

Coleslaw

Coleslaw 是用 Common Lisp 编写的静态网站生成器。它是一个轻量级和灵活的工具，可以轻松地创建简单或复杂的博客。使用 Coleslaw 构建静态博客，第一步是在计算机上安装它，可以按 Coleslaw 网站上的说明进行安装。

安装 Coleslaw 后，可以通过在终端中运行coleslaw init命令来创建一个新的博客。

下一步是自定义您的博客。Coleslaw 使用 Google 闭包模板生成博客的 HTML 页面，因此需要创建或修改模板以满足需求。Coleslaw 附带了两个默认模板，您可以将其用作起点，也可以从头开始创建自己的模板。

要为您的博客创建新文章，可以运行coleslaw post命令。这将在“posts”目录中创建一个新的 Markdown 文件，您可以编辑该文件以撰写文章。您可以使用 Markdown 语法向文章中添加图像、链接和其他内容。

当您准备好发布您的博客时，可以运行coleslaw deploy命令。这将为您的博客生成HTML页面，并将它们放置在deploy目录中。然后，您可以将这些文件上传到 Web 服务器或 GitHub 托管服务，以使你的博客在互联网上可用。

总之，使用 Coleslaw 构建一个静态博客是一个简单的过程，只需要几个基本步骤。使用 Coleslaw，你可以创建一个个性化和专业的博客，可以轻松定制和更新。那么，为什么不尝试一下，今天就开始与世界分享你的想法吧！
